## 1. Vue d'ensemble et accès

### 1.1 À quoi sert le module IA

Le module **Assistant Intelligent (IA)** intègre **Claude (Anthropic)** dans Constructo pour vous aider sur de nombreuses tâches :

- **Chat conversationnel** avec un assistant spécialisé construction
- **Profils d'experts** : estimateur, chef de chantier, comptable, architecte, etc.
- **Analyse de documents** (PDF, plans, devis concurrents) : extraction automatique
- **Analyse d'images** (photos chantier) : description, détection anomalies, OCR
- **Catégorisation de notes** (projets, dossiers) selon 10 catégories métier
- **Résumés automatiques** (notes longues, rapports)
- **Génération de devis** à partir de paramètres (type, superficie)
- **Calculs et suggestions** (prix marché, normes, conformité)

Le module utilise un **système de crédits IA** payant (selon usage tokens).

### 1.2 Concepts-clés

| Concept | Description |
|---|---|
| **Conversation** | Session de chat avec l'IA (contexte conservé) |
| **Profil expert** | Persona spécialisée (estimateur, plombier, etc.) |
| **Documents joints** | Fichiers contexte (plans, devis) ajoutés à la conversation |
| **Crédit IA** | Unité de facturation (basée sur tokens consommés) |
| **Quota mensuel** | Limite de crédits disponibles selon votre plan |
| **Modèle Claude** | claude-opus-4-7 (par défaut, le plus puissant) |
| **Markup** | Marge ajoutée au coût Anthropic (typique 30 %) |

### 1.3 Accès

- **Sidebar** → **Assistant IA** (icône robot)
- **URL** : `/assistant-ia`
- **IA contextuelle** : disponible aussi dans Devis, Dossiers (via boutons « Catégoriser IA », « Analyser photo », etc.)

### 1.4 Permissions

- **Tous les utilisateurs authentifiés** peuvent utiliser l'IA
- **Solde de crédits** doit être suffisant
- **Module IA activé** pour le tenant (par administrateur)
- **Quota mensuel** peut limiter l'usage

## 2. Interface — Assistant et conversations

### 2.1 Page Assistant IA (`/assistant-ia`)

Layout :
```
+--------------------------------------------------------------+
| Profil : [v Estimateur]   [Crédits : 4 250 / 10 000]        |
+----------------+---------------------------------------------+
| CONVERSATIONS  | CHAT (conversation active)                  |
| (sidebar)      |                                             |
|                | [USER] Quel est le prix moyen du m² de      |
| - Reno cuisine |        plancher de bois franc à Montréal ?  |
| - Estimat. M.  |                                             |
| - Devis garage | [IA] Le prix moyen actuel à Montréal pour   |
| ...            |      un plancher de bois franc se situe     |
|                |      entre 8 et 14 $/pi²...                 |
| [+ Nouvelle]   |                                             |
+----------------+---------------------------------------------+
| [Documents joints (drag & drop)]                            |
| [Saisir votre message...]                       [Envoyer]   |
+--------------------------------------------------------------+
```

### 2.2 Sélecteur de profil expert

Dropdown en haut. Profils disponibles :
- **Généraliste** (par défaut)
- **Estimateur** (calculs prix, soumissions)
- **Chef de chantier** (planification, sécurité)
- **Architecte** (plans, normes du Code du bâtiment)
- **Ingénieur** (structure, calculs)
- **Comptable** (TPS/TVQ, normes comptables)
- **Charpentier** / **Plombier** / **Électricien** (métiers)
- **Avocat construction** (contrats, hypothèques légales)
- **Inspecteur** (conformité)

### 2.3 Compteur de crédits

Affiché en haut à droite : **Crédits utilisés / Crédits disponibles**.

Couleurs :
- Vert : > 50 % restants
- Ambre : 20-50 %
- Rouge : < 20 % (alerte)

### 2.4 Liste des conversations (sidebar)

- Nom auto-généré (premier message tronqué)
- Date dernière activité
- Bouton « + Nouvelle conversation »
- Recherche dans les conversations
- Suppression (icône poubelle)

### 2.5 Zone de chat

#### Messages utilisateur (droite, fond bleu)
Texte simple ou avec pièces jointes (icône fichier).

#### Réponses IA (gauche, fond gris)
Texte formaté (markdown rendu : titres, listes, tableaux, code).
Boutons : **Copier**, **Régénérer**, **Continuer**.

### 2.6 Documents joints

Glisser-déposer ou cliquer pour ajouter des fichiers (PDF, JPG, PNG) qui seront utilisés comme contexte par l'IA pour cette conversation.

Exemples d'usage :
- Joindre un plan PDF puis demander « Calcule la superficie totale »
- Joindre photo chantier puis demander « Décris ce que tu vois »
- Joindre devis concurrent puis demander « Compare avec ce devis et donne-moi les écarts »

### 2.7 Champ de saisie

- Multiligne (Shift+Enter pour nouvelle ligne)
- Enter pour envoyer
- Bouton « Envoyer » avec icône avion

### 2.8 Modale Crédits & Usage

Accessible via icône statistiques. Contient :
- **Solde actuel**
- **Quota mensuel** (selon plan)
- **Consommation par jour** (graphique 30 derniers jours)
- **Consommation par mois** (graphique 12 derniers mois)
- **Top usage** (par module : Assistant, Catégorisation, Analyse doc, etc.)
- **Coût détaillé** (tokens entrée + sortie)

### 2.9 Vue mobile

Sidebar conversations en menu hamburger, chat en plein écran, attache fichiers via bouton dédié.

## 3. Workflows pas-à-pas

### 3.1 Démarrer une nouvelle conversation
1. `/assistant-ia` → bouton **« + Nouvelle conversation »**
2. (Optionnel) Choisir un profil expert dans le dropdown
3. Saisir votre question dans le champ de saisie
4. Cliquer **« Envoyer »** ou Enter
5. Attendre la réponse de l'IA (généralement 2-10 secondes)

### 3.2 Changer de profil expert en cours de conversation
1. Dropdown profil → choisir nouveau profil
2. La nouvelle question utilisera le profil sélectionné
3. L'IA garde le contexte de la conversation

### 3.3 Joindre un document à la conversation
1. Glisser-déposer le fichier (PDF/JPG/PNG) sur la zone de chat
2. OU cliquer sur l'icône trombone → sélectionner
3. Le fichier apparaît au-dessus du champ de saisie
4. Saisir votre question liée au document
5. Cliquer Envoyer

### 3.4 Analyser un plan PDF
1. Joindre le PDF du plan
2. Profil : **Architecte** ou **Estimateur**
3. Demander : « Calcule la superficie totale » ou « Liste les pièces et leurs dimensions »
4. L'IA extrait les données et répond

### 3.5 Analyser une photo de chantier
1. Joindre la photo
2. Profil : **Inspecteur** ou **Chef de chantier**
3. Demander : « Décris ce que tu vois et identifie les anomalies »
4. L'IA décrit le contenu et signale les risques

### 3.6 Extraire les données d'un devis concurrent
1. Joindre le PDF du devis
2. Profil : **Estimateur**
3. Demander : « Extrais les lignes et fais-moi un tableau Description / Qté / Prix »
4. L'IA structure les données

### 3.7 Demander un calcul de prix marché
1. Profil : **Estimateur**
2. Question : « Quel est le prix moyen actuel pour [produit/service] à [ville] ? »
3. L'IA répond avec une fourchette de prix typique

### 3.8 Vérifier la conformité au Code du bâtiment
1. Profil : **Architecte** ou **Inspecteur**
2. Question : « Quelles sont les exigences du CNB pour [élément] dans [type de bâtiment] ? »
3. L'IA répond selon le Code national du bâtiment et le Code du bâtiment du Québec

### 3.9 Calculer une cotisation CCQ ou CNESST
1. Profil : **Comptable** ou **Avocat construction**
2. Question : « Calcule la cotisation CCQ pour [N heures] @ [taux] »
3. L'IA répond avec le détail du calcul

### 3.10 Rédiger un email client
1. Profil : **Généraliste**
2. Question : « Rédige-moi un email pour informer un client d'un retard de chantier de 2 semaines »
3. L'IA propose un texte professionnel à adapter

### 3.11 Synthétiser un rapport
1. Joindre le rapport (PDF ou copier-coller le texte)
2. Demander : « Résume ce rapport en 5 points clés »
3. L'IA produit un résumé structuré

### 3.12 Catégoriser une note projet (depuis module Projets)
Depuis la fiche du projet → Note → bouton **« Catégoriser IA »** (voir manuel **Projets** section 3.10).

### 3.13 Catégoriser une note dossier (depuis module Dossiers)
Idem, depuis le dossier (voir manuel **Dossiers** section 3.7).

### 3.14 Analyser une photo de note dossier
Depuis la note avec photo → bouton **« Analyser photo IA »** (voir manuel **Dossiers** section 3.8).

### 3.15 Résumer plusieurs notes (Dossiers)
Depuis le dossier → bouton **« Résumé IA »** (voir manuel **Dossiers** section 3.9).

### 3.16 Générer un devis complet via IA
Depuis le module **Devis** → bouton **« IA — Générer soumission »** (voir manuel **Devis** section 3.16).

### 3.17 Scanner une facture par IA
Depuis le module **Factures** → bouton **« Scan IA »** (voir manuel **Factures** section 3.3).

### 3.18 Régénérer une réponse
Si la réponse ne convient pas, cliquer **« Régénérer »** sous le message IA. Une nouvelle réponse est générée (consomme à nouveau des crédits).

### 3.19 Continuer une réponse tronquée
Si l'IA s'arrête à mi-réponse (limite tokens), cliquer **« Continuer »** pour suivre la suite.

### 3.20 Copier une réponse
Bouton **« Copier »** sous le message IA → texte copié dans le presse-papiers.

### 3.21 Supprimer une conversation
Sidebar → icône poubelle à côté de la conversation → confirmer.

### 3.22 Consulter votre usage et coût
1. Cliquer sur l'icône statistiques (en haut à droite)
2. Vue : Solde, Consommation par jour/mois, Top usage par module
3. Détail : tokens entrée + sortie, coût total

### 3.23 Recharger des crédits IA
**Réservé aux administrateurs** :
1. Modules → Configuration → IA → Recharger crédits
2. Saisir montant à acheter
3. Paiement Stripe
4. Crédits ajoutés au solde

## 4. Référence — Profils, modèles et tarification

### 4.1 Profils experts disponibles (10+)

| Profil | Spécialité | Usages typiques |
|---|---|---|
| **Généraliste** | Polyvalent | Questions générales, rédaction |
| **Estimateur** | Calculs prix, soumissions | Estimations, marges, prix marché |
| **Chef de chantier** | Planification, sécurité | Gestion équipes, échéancier |
| **Architecte** | Plans, normes du Code | Conformité CNB, esthétique |
| **Ingénieur** | Structure, calculs | Charges, dimensionnement |
| **Comptable** | TPS/TVQ, normes comptables | Calculs taxes, écritures |
| **Charpentier** | Métier charpente | Conseils techniques bois |
| **Plombier** | Métier plomberie | Conseils techniques eau |
| **Électricien** | Métier électricité | Conseils techniques électricité |
| **Avocat construction** | Contrats, hypothèques légales | Aspects juridiques |
| **Inspecteur** | Conformité | Vérifications techniques |

### 4.2 Modèle Claude par défaut

**claude-opus-4-7** (Anthropic) :
- Modèle le plus avancé d'Anthropic
- Excellent en raisonnement, code, analyse
- Vision (image, PDF) supportée
- Contexte 200K tokens
- Multi-langue (français parfait)

### 4.3 Tarification (indicative)

Coût Anthropic + markup 30 % :

| Type | Coût Anthropic | Coût Constructo (avec markup) |
|---|---|---|
| Tokens entrée | $0,015 / 1K | $0,0195 / 1K |
| Tokens sortie | $0,075 / 1K | $0,0975 / 1K |

**Exemple** : conversation moyenne (5 échanges, 10 000 tokens entrée + 5 000 tokens sortie) ≈ **0,68 $ CAD**.

### 4.4 Quotas par plan

| Plan | Crédits/mois | Coût /mois |
|---|---|---|
| Starter | 10 000 | inclus |
| Pro | 50 000 | inclus |
| Business | 200 000 | inclus |
| Enterprise | Illimité | sur devis |

> Les crédits supplémentaires sont disponibles à l'unité (paiement à la consommation).

### 4.5 Limite de taille des fichiers joints

| Type | Taille max |
|---|---|
| PDF | 32 Mo |
| Image (JPG/PNG) | 20 Mo |
| Document Office (DOCX, XLSX) | 50 Mo |
| Total par conversation | 100 Mo |

### 4.6 Catégories IA pour notes (10)

Voir **Module Projets section 4.6** ou **Module Dossiers section 4.8** :
Technique, Sécurité, Budget, Planning, Qualité, Communication, Environnement, RH, Approvisionnement, Autre.

### 4.7 Score d'importance pour notes

Décimal 0-1 :
- 0,8-1,0 : critique
- 0,5-0,79 : important
- 0,3-0,49 : moyen
- 0,0-0,29 : faible

### 4.8 Limites système

| Élément | Limite |
|---|---|
| Conversations actives | Pas de limite stricte |
| Messages par conversation | 200 (ensuite, ancienneté tronquée du contexte) |
| Documents par conversation | 5 max |
| Vitesse de réponse | 2-15 secondes selon complexité |
| Concurrent requests | 5 par utilisateur |

## 5. Intégrations, cas particuliers et FAQ

### 5.1 Intégrations avec les autres modules

| Module | Fonction IA | Manifestation |
|---|---|---|
| **Projets** | Catégorisation notes | Bouton « Catégoriser IA » sur chaque note |
| **Dossiers** | Catégorisation, analyse photo, résumé | Boutons dédiés sur les notes |
| **Devis** | Analyse documents, génération auto, chat | Boutons dans constructeur |
| **Factures** | Scan facture (extraction PDF) | Bouton « Scan IA » |
| **Comptabilité** | Suggestion d'écritures | À venir |
| **Tous modules** | Assistant contextuel | Disponible partout via Sidebar |

### 5.2 Cas particuliers

- **Profil non spécialisé pour la question** : changer de profil et régénérer
- **Fichier trop gros** : compresser ou découper en plusieurs envois
- **Réponse incorrecte** : régénérer ou reformuler la question
- **Crédits épuisés** : recharger ou attendre le mois suivant (quota)
- **Conversation très longue** : créer une nouvelle conversation pour repartir avec contexte propre
- **Contenu confidentiel** : les conversations sont stockées dans votre tenant. Anthropic ne conserve pas les données après traitement (selon politique privacy)
- **Demande hors périmètre** : si vous demandez des conseils juridiques très pointus, l'IA recommande de consulter un avocat

### 5.3 Astuces

- **Choisir le bon profil** : améliore considérablement la pertinence des réponses
- **Joindre des documents** : l'IA est plus précise avec contexte (plans, devis, photos)
- **Questions précises** : « Calcule X pour Y » mieux que « Donne-moi des infos »
- **Demander un format** : « Réponds-moi sous forme de tableau » ou « Liste à puces »
- **Itérer** : ne pas hésiter à reformuler si la réponse n'est pas idéale
- **Crédits** : surveiller régulièrement le solde pour éviter l'interruption en cours de tâche
- **Conversations dédiées** : une conversation par sujet pour garder le contexte propre
- **Catégorisation IA** : utiliser pour trier rapidement les notes massives

### 5.4 FAQ

**Q : Mes données sont-elles partagées avec Anthropic ?**
R : Les requêtes sont envoyées à l'API Anthropic pour traitement. Selon la politique de privacy d'Anthropic, les données ne sont pas conservées après traitement et ne servent pas à entraîner les modèles.

**Q : Puis-je utiliser l'IA en français du Québec ?**
R : Oui, Claude est excellent en français incluant les expressions québécoises et le vocabulaire de construction local.

**Q : Comment réduire ma consommation de crédits ?**
R : Conversations courtes et précises, profils adaptés (pas de Généraliste pour des calculs précis), éviter les fichiers volumineux inutiles, utiliser le bouton Régénérer avec parcimonie.

**Q : Le module fonctionne-t-il hors ligne ?**
R : Non, l'IA nécessite une connexion à l'API Anthropic.

**Q : Puis-je télécharger l'historique de mes conversations ?**
R : Pas dans cette version. Les conversations restent en base et sont consultables.

**Q : Comment l'IA a-t-elle accès à mes données ?**
R : Pour les fonctions contextuelles (catégorisation, analyse), seules les données nécessaires (titre, contenu d'une note, ligne d'un devis) sont envoyées. L'IA n'a pas accès à votre base complète.

**Q : Puis-je personnaliser un profil expert ?**
R : Pas dans cette version. Les profils sont prédéfinis. Possibilité d'ajouter des profils personnalisés à venir.

**Q : Le coût des crédits varie-t-il selon le profil ?**
R : Non, le coût dépend uniquement du nombre de tokens entrée/sortie, pas du profil.

**Q : Puis-je voir le coût exact de chaque message ?**
R : Vue Usage → détail par conversation. Coût par message visible.

**Q : Comment savoir si l'IA s'est trompée ?**
R : Toujours vérifier les calculs, prix, dates suggérés. L'IA est un assistant, pas un oracle. Les réponses sont à valider.

**Q : Puis-je utiliser l'IA pour générer du contenu pour mon site web ?**
R : Oui, mais respecter les conditions d'usage commercial d'Anthropic.

**Q : Le module supporte-t-il la dictée vocale ?**
R : Pas nativement. Utiliser la dictée vocale du navigateur ou de votre OS.

### 5.5 Limites connues

- Modèle Claude figé (pas de choix entre Sonnet/Opus dans cette version)
- Pas de profils personnalisables
- Pas de mode hors ligne
- Pas d'export historique conversations
- Pas de dictée vocale native
- Limite 5 documents par conversation
- Quotas mensuels rigides (pas de roll-over si non utilisés)
- Pas de partage de conversation entre utilisateurs (privées)
- Coût en CAD avec taux de change figé (mise à jour périodique)
